# 🎉 MoyuWindows v1.0.0 发布说明

## 📦 下载

- **Windows x64**: `Moyu-win-x64.zip` (约 70MB 压缩后)
- **Windows ARM64**: `Moyu-win-arm64.zip`

## ✨ 主要功能

### 学习模式
- **记忆模式**: 单词卡片学习，显示单词、音标、翻译、例句
- **选择题模式（中文→英文）**: 看中文选英文单词
- **选择题模式（英文→中文）**: 看英文选中文翻译 ⭐ 新功能
- **拼写模式**: 看中文输入英文单词 ⭐ 新功能

### 词库支持
- 四级核心词汇
- 六级核心词汇
- 考研词汇
- 托福词汇
- 雅思词汇
- GRE 词汇
- SAT 词汇
- 日语五十音

### 核心特性
- ✅ 系统托盘常驻
- ✅ 全局热键 (Ctrl+Shift+M)
- ✅ TTS 真人发音
- ✅ 收藏夹和错词本
- ✅ 学习统计和成就系统
- ✅ 深色/浅色主题
- ✅ Windows 11 圆角设计 ⭐
- ✅ 单文件 exe 分发 ⭐
- ✅ 自定义词库导入
- ✅ 数据导出备份

## 🚀 快速开始

1. 下载 `Moyu-win-x64.zip`
2. 解压到任意目录
3. 运行 `Moyu.exe`
4. 开始学习！

**特点：**
- 单文件 exe（188MB）
- 无需安装 .NET 运行时
- 数据库和图标已内置
- 开箱即用

## 📝 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+M` | 全局热键：显示/隐藏窗口 |
| `ESC` | 隐藏窗口 |
| `Enter` | 提交答案/下一题 |
| `1-4` / `A-D` | 选择题快速选择 |
| `S` | 朗读单词 |
| `F` | 收藏/取消收藏 |
| `Space` / `→` | 下一个单词 |

## 🎯 新功能亮点

### 拼写模式 ⭐
真正的主动学习，通过输入单词来加深记忆：
- 显示中文释义和音标
- 输入英文单词
- 实时验证（忽略大小写）
- 正确/错误反馈
- 错误时显示正确答案
- 支持跳过

### 英文选中文模式 ⭐
反向学习，提高理解能力：
- 显示英文单词
- 选择正确的中文翻译
- 4 个选项
- 即时反馈

### Windows 11 设计 ⭐
现代化界面：
- 更大的圆角（16px 主窗口）
- 柔和的阴影效果
- 符合 Fluent Design 规范
- 视觉效果更舒适

### 单文件分发 ⭐
极致便利：
- 只需一个 exe 文件
- 数据库已嵌入（27MB）
- 图标已嵌入
- 真正的开箱即用

## 🛠️ 系统要求

- Windows 10 1809 或更高版本
- 无需安装 .NET 运行时
- 约 200MB 磁盘空间

## 📊 技术细节

- **框架**: .NET 8.0 + WPF
- **数据库**: SQLite
- **发布方式**: 单文件自包含
- **文件大小**: 约 188MB（包含运行时和数据库）

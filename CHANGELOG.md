# 更新日志

## [未发布] - 2026-01-08

### ✨ 新增功能

#### 拼写模式（重大更新）⭐
- **拼写练习页面**：新增 SpellingPage，用户可以通过输入单词来测试记忆
  - 显示中文释义和音标
  - 支持 TTS 发音
  - 实时答案验证（忽略大小写）
  - 正确/错误反馈显示
  - 错误时显示正确答案
  - 支持跳过功能
  - 快捷键支持（Enter 提交，Esc 跳过）

#### 测试模式完整支持
- **中文选英文模式**：显示中文释义，选择英文单词（原有）
- **英文选中文模式**：显示英文单词，选择中文翻译（新增）
- **拼写模式**：显示中文释义，输入英文单词（新增）
- 在设置中可以切换测试模式
- 学习时自动使用选定的测试模式

#### Windows 11 圆角设计 ⭐
- 主窗口圆角从 12px 增大到 16px
- 全局圆角尺寸优化：
  - RadiusSM: 4px → 6px
  - RadiusMD: 8px → 10px
  - RadiusLG: 12px → 14px
  - RadiusXL: 16px → 20px
- 符合 Windows 11 Fluent Design 风格
- 阴影效果优化，更柔和

#### 单文件 EXE 分发 ⭐
- 图标嵌入到 exe 中
- 数据库嵌入到 exe 中
- 真正的单文件分发（约 100MB）
- 开箱即用，无需额外文件
